 # 🎵 Spotify Clone

A responsive Spotify-inspired music streaming interface built using **HTML** and **CSS**. This project focuses on recreating the modern Spotify UI while improving front-end development skills. JavaScript functionality will be added in future updates.

> **🚧 Project Status:** Under Development
> The current version includes the user interface only. Interactive features and music playback will be implemented after learning JavaScript.
